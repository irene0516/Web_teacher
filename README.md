# Web_teacher
網頁練習(老師個人網站，本網站為教學使用，擷取自亞洲大學資工系網站)
此網頁為老師個人網站練習，並包含登入修改網頁的功能。主要使用的技術有:
* PHP
* HTML
* CSS
* JavaScript
* MySQL

此外，使用jQuery以及Bootstrap完善網頁功能與設計。
